A web-based augmented-reality version of the card game W.A.R. The hosted
demo is gone (Heroku killed free dynos in 2022), but the source is still on
GitHub.

Mostly a sketch. A way to learn what casual browser-AR could feel like in
2016, before AR libraries were a thing you could `npm install`. The hosted
demo is gone with the Heroku free tier; the source is still there. Worth
keeping for the question, not the answer.
